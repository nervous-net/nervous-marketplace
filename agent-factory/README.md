# Agent Factory

Create teams of persistent, personality-driven sub-agents for any project. Describe what you need help with and the factory designs, scaffolds, and deploys your agent team.

## Skills

### `/create-agents`

The primary workflow. Runs a 5-phase interview-driven process to design and scaffold a full agent team.

### `/add-agent`

Extend an existing team. Detects your current roster, interviews about gaps, and scaffolds new agents.

### `/upgrade-agent`

Three upgrade paths: MCP Server, Cron Schedule, or Multi-Project Promotion.

### `/hire-agent`

Hire an agent from another project or global scope. Discovers agents via a central registry, matches by intent, and copies with light adaptation.

## Installation
